HTML & CSS References

transition

A shorthand property for all the transition-* properties

@keyframes

Specifies the animation code

transform

Applies a 2D or 3D transformation to an element

animation

A shorthand property for all the animation-* properties

animation-direction

Specifies whether an animation should be played forwards,

backwards or in alternate cycles

justify-content

Specifies the alignment between the items inside a flexible

container when the items do not use all available space

opacity

Sets the opacity level for an element

z-index

Sets the stack order of a positioned element

overflow-x

Specifies whether or not to clip the left/right edges of the content,

if it overflows the element's content area
